Starstar Project Information

This project was a collaborative commission for a jcink roleplaying website, Sinister Haven. This project was completed by myself and Jessica Teichman. Starstar is a custom discord bot created for Sinister Haven. This bot has a lot of custom commands that are specific to the website and fandom. For now it is only used on the Sinister Haven discord server, but we plan to make it available to the general public in the near future.

Project Goals

  • Create a custom discord bot for Sinister Haven, with a wide variety of custom commands
  • Starstar must be able to complete important server functions such as welcoming new members, answering questions, ect.
  • Create fun games for site members to play with StarStar
  • Create a bot that will be able to be used in other servers eventually

Starstar Commands

  • /TALK -- This command allows StarStar to answer member questions through their speakers. When asked a question using this command, Starstar sends the answer to the chat as well as speaking it aloud.
  • /CAT -- This command generates a random character that roleplayers can adopt as their own character. It colors in a lineart image with a random choice of colors and markings, so that every cat generated is unique. It also generates a unique two part name, description, and set of character traits.
  • /RCL -- This command stands for Rock, Claw, Leaf. It is a warrior cat version of rock paper scissors that site members can play with Starstar.
  • /ALASKA -- These two commands show a random picture of a cat named Alaska. Alaska is a cat owned by one of the Sinister Haven admins and the unofficial mascot of the website. This command was created so site members could see pictures of Alaska at any time.
